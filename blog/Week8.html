My "Aha" moment of the week:

I was using the console.log to see if my code was working, and when it wasn't right away but would a second later, I realized I needed to use async await, which is the first time I've been able to use it understanding exactly how it works.

One thing I learned about coding this week:

I have known about this and thought I was implementing it before, but I hadn't fully grasped exactly how small I need to break down the steps in my pseudocoding. I was feeling overwhelmed by thinking of the next task ahead without realizing that it was actually 10 or so small steps, and each of those steps is much more surmountable than all 10 put together! I also learned the importance of checking (via console.log mostly) every single little step to make sure it's working, even when the steps will be interdependent later on.
